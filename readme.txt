# MotoVida - TP Página Web

## Propósito
Este proyecto es una página web desarrollada como trabajo práctico a media cursada. Representa un sitio para "MotoVida", un negocio dedicado a la compra y venta de motos, con servicios, reseñas y un formulario de contacto. Utiliza HTML5 semántico, CSS con Flexbox y Grid, y Formspree para el formulario. La sección de contacto está en un archivo HTML separado (`pages/contacto.html`).
